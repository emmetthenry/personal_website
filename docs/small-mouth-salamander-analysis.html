Small-Mouth Salamander Sightings in the United States

Emmett Henry

Overview

Small-mouth salamanders (Ambystoma texanum) are an endangered species of salamander native to the United States. Tracking their location and population can give insights into the health of the species.

Data & methods

I used data from iNaturalist, and filtered for Small-Mouthed Salamander sightings in the United States.

# load packages and data here
library(ggplot2)
library(tmap)
library(sf)
library(stringr)

salamanders <- read.csv("salamander_data_cleaned_CSV.csv")

Analysis

# count sightings by year for 2020 onwards

#initialize count variables
count2020 <- 0
count2021 <- 0
count2022 <- 0
count2023<- 0
count2024 <- 0
count2025 <- 0

# access date column
dates <- salamanders$observed_on

# loop through dates, access year, increment appropriate variable
for (sighting_date in dates) {
  if (str_sub(sighting_date, -4, -1) == "2020") {
    count2020 <- count2020 + 1
  } else if (str_sub(sighting_date, -4, -1) == "2021") {
    count2021 <- count2021 + 1
  } else if (str_sub(sighting_date, -4, -1) == "2022") {
    count2022 <- count2022 + 1
  } else if (str_sub(sighting_date, -4, -1) == "2023") {
    count2023 <- count2023 + 1
  } else if (str_sub(sighting_date, -4, -1) == "2024") {
    count2024 <- count2024 + 1
  } else if (str_sub(sighting_date, -4, -1) == "2025") {
    count2025 <- count2025 + 1
  }
}


# Combine data into a single data frame
n_sightings_df <- data.frame(
  year_obs = as.character(seq(from = 2020, to = 2025, by = 1)),
  number_obs = c(count2020, count2021, count2022, count2023, count2024, count2025)
)

# Figure
n_sightings_fig <- ggplot() +

  # Columns
  geom_col(
    data = n_sightings_df,
    aes(
      x = year_obs,
      y = number_obs
    )
  )

# draw figure
n_sightings_fig

Visualization

  theme_minimal() +
  ggtitle("Small-Mouthed Salamander iNaturalist Sightings")

Results

Sightings of Small-Mouthed Salamanders have been steadily increasing from 2020 to 2024. There was nearly a doubling in overall iNaturalist sightings during this period, with 236 sightings in 2020 and 411 sightings in 2024.However, although 2025 is not yet over, this year could be predicted to have a lower sighting of small-mouthed salamanders. Since they are most active during their breeding season from February to March and with only two months left in the year, 2025’s count of 328 is unlikely to reach 2024’s count of 411 sightings.

Reflection

This data is limited by its open-source nature, as all sightings are provided by volunteer users. There is no set number of volunteers, nor is there a set number of observations that each volunteer reports per year. To further isolate trends in the species itself, the number of small-mouthed salamander sightings per year could be compared against overall iNaturalist sightings for each year. After that, trends pointing to causation could be studied, using data on the salamanders’ habitats across each year and data on proximity to potential threats.
